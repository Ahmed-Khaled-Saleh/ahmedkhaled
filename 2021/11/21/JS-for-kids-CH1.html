Intro (Computers what they are about)

Computers are actually pretty dumb, deep down. They can do amazing things but they don't actially do it. Computers obey the orders of the electrcial signals of 0s and 1s. Computers can only do exactly what we humans tell them to do. But we don't tell the Computer what to do by using a pattern of 0s and 1s but with a set of languages called PLs. PLs are like english, they have Grammer and Vocabulary . One such language is JS .

Meet JS

JS is used to write programs that run on Web browsers. JS can control how a web page looks, or behave (like what action to do when a user press certain key or move the mouse). **ex : scrolling down in Twitter brings more tweets to life .

Resons to learn JS

1- it's easier and morefun to learn than many other PLs. 2- the prerequists tools are very minmal (only a web browser). (every web browser comes with JS interpreter). 3- Portability of JS programs (JS runs on any device, as long as you have a browser installed).

Syntax :

Is the language rules for combining symbols and words to create working programs . (eg, in English, we know that the structure of the sentence is subject + verb + Object). like english, JS has its own syntax and rules of combination. It's easeier to know theam gradually rather than mentioning them here.

Comments :

Comments are ignored by JS interpreter, they are there for documenting purposes. Comments have two puropses : 1- For other programmers to understand what the code does . 2- For yourself when you return to work on your program after a while. There are two types of comments : 1- Single line //this is a comment 2- Multiline / this is a comment/
